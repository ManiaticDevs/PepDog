### Roblox Clone in LWJGL2

## Java Version is 8

Engine which aims to implement as much features as the original Roblox 2006-2010 client. (OPENGL)

# CONTROLS
## Player:
- WASD - move
- SPACE - jump
- V - noclip (hold)
- I - Zoom In
- O - Zoom out

## Fly Cam:
- WASD - move
- L.CONTROL - Down
- SPACE - Up

## Misc.
- Q - FlyCam
- Left Shift - Sprint
- J - Switch maps via input box (must match map name and also like limited use??)
